## JavaScript Basic Logic Programs

This section covers a set of beginner-friendly JavaScript programs focused on building logic and problem-solving skills.

---

## Even or Odd Checker

This program determines whether a given number is even or odd.

- Uses the modulus (`%`) operator  
- Returns **"Even"** if divisible by 2  
- Otherwise returns **"Odd"**  

---

## Result Evaluator (Pass or Fail)

This program evaluates a student's result based on marks.

- Validates input (marks should be between 0 and 100)  
- Returns **"Pass"** if marks are 50 or above  
- Returns **"Fail"** if marks are below 50  
- Handles invalid input gracefully  

---

## Maximum Finder (Largest of Three Numbers)

This logic finds the largest number among three values.

- Compares numbers step by step  
- Updates the maximum value dynamically  
- Returns the highest number  

---

## Accumulator (Sum from 1 to N)

This program calculates the sum of numbers from 1 up to a given number.

- Uses a loop to accumulate values  
- Useful for understanding iterative addition  
- Example: Sum of first 5 numbers → 15  

---

## Multiplication Table Generator

Generates a multiplication table for a given number.

- Iterates from 1 to 10  
- Displays results in a readable format  
- Helps understand loops and number patterns  

---

## Digit Counter (Number Length Finder)

Counts the number of digits in a number.

- Handles zero as a special case  
- Works with both positive and negative numbers  
- Uses division to reduce the number step by step  

---

## Reverse Number

Reverses the digits of a given number.

- Extracts digits one by one  
- Builds the reversed number  
- Example: 12345 → 54321  

---

## Factorial Engine

Calculates the factorial of a number.

- Multiplies all integers from 1 to the given number  
- Example: 5! = 120  
- Commonly used in mathematics and algorithms  

---

## Prime Validator

Checks whether a number is prime.

- A prime number is divisible only by 1 and itself  
- Uses logical checks to determine validity  
- Important concept in number theory  

---

## Pattern Builder (Star Triangle)

Prints a triangle pattern using stars (`*`).

- Uses nested loops  
- Each row increases the number of stars  
- Helps understand pattern logic and loop control  

---

- Conditional statements  
- Loops (for, while)  
- Functions and logic building  
- Mathematical computations  
- Pattern generation
